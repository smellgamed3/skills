# Skills Collection

My collection of useful Claude Code skills.

## What are Skills?

Skills are reference guides for proven techniques, patterns, or tools that help Claude Code instances work more effectively.

## Available Skills

### outline-cli
Outline API interaction skill for managing documents, collections, and searches.

**Features:**
- Document CRUD operations
- Collection management
- Search functionality
- Authentication (API Key + OAuth)
- Error handling and security best practices

**Usage:**
- Direct API calls with curl examples
- Integration with outline-cli tool
- Self-hosted instance support

## Installation

Each skill can be installed by creating a symbolic link to your Claude skills directory:

```bash
ln -s ~/skills/skill-name/SKILL.md ~/.claude/skills/skill-name-SKILL.md
```
